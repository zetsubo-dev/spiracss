# SpiraCSS Stylelint Rule Reference

This directory contains OK / NG / Why references for each rule. For installation and configuration, see [Stylelint](../stylelint.md).

## Rules

- [spiracss/class-structure](class-structure.md): naming, parent-child structure, section structure
- [spiracss/property-placement](property-placement.md): property placement (container / item / internal)
- [spiracss/interaction-scope](interaction-scope.md): interaction section placement and structure
- [spiracss/interaction-properties](interaction-properties.md): transition / animation placement and constraints
- [spiracss/keyframes-naming](keyframes-naming.md): `@keyframes` placement and naming
- [spiracss/pseudo-nesting](pseudo-nesting.md): pseudo-class / pseudo-element nesting
- [spiracss/rel-comments](rel-comments.md): `@rel` comments and parent-child links
