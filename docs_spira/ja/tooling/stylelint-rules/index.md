# SpiraCSS Stylelint ルール詳細

各ルールの OK / NG / 理由をまとめたリファレンスです。導入・設定は [Stylelint](../stylelint.md) を参照してください。

## ルール一覧

- [spiracss/class-structure](class-structure.md): 命名・親子構造・セクション構造

- [spiracss/interaction-scope](interaction-scope.md): interaction セクションの位置と構造
- [spiracss/interaction-properties](interaction-properties.md): transition / animation の配置と制約
- [spiracss/keyframes-naming](keyframes-naming.md): `@keyframes` の配置と命名
- [spiracss/pseudo-nesting](pseudo-nesting.md): 疑似クラス / 疑似要素のネスト
- [spiracss/rel-comments](rel-comments.md): `@rel` コメントと親子リンク
